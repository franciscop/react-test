Render and assert

This is the most basic functionality of this library; it allows you to render a React component and make assertions against it with Jest or any other library you use. You can either use the provided methods to access properties like .text(), or grab the first node with .first() and then use native DOM methods. 

import $ from "react-test"

it("renders a simple paragraph", async () => {
  const paragraph = $(<p>Hello</p>);
  expect(paragraph.text()).toBe("Hello");
  expect(paragraph.first().nodeName).toBe('P');
  // ...
});
